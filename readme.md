# 🧠 Skin Disease Detection System

Deep learning-based skin disease detection system using **MobileNetV2** trained on the HAM10000 dataset for multi-class classification of skin lesions.

## 📌 Features

- Multi-class skin disease classification
- Lightweight and efficient MobileNetV2 model
- Image preprocessing and augmentation
- Disease prediction with high accuracy
- Scalable for web deployment

## 📂 Dataset

Dataset used: HAM10000 (Human Against Machine with 10000 training images)

🔗 https://www.kaggle.com/datasets/surajghuwalewala/ham1000-segmentation-and-classification

## 🧠 Model Architecture

- Base Model: MobileNetV2 (Transfer Learning)
- Fine-tuning applied
- Fully connected layers for classification
- Softmax activation for multi-class output

## ⚙️ Tech Stack

- Python
- TensorFlow / Keras
- OpenCV
- NumPy, Pandas
- Matplotlib
